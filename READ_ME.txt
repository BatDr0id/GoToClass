All files used for web server and database are in 
website_datatbase_files folder

All the php files get put in the root of the website
and urls in java stringrequest classes must be changed.

"images" folder must be put in the same path as php files.
Folder must be named “images”. This will contain all profile
Images for all users. Paths to the images stored in data may 
Have to change to fit your websites model

create_database.sql file is used to create and input
current mockup tables and data
